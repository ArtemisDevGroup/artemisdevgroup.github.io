What is Artemis?

Project Artemis is a tool developed by a small team of reverse engineers and programmers aiming to make playing a legacy build of Rainbow Six: Siege as easy, seamless and enjoyable as possible.

Artemis provides a number of modifications for all target audiences. It provides everything from fun gamemodes and freely configurable equipment to full fetched cheat features such as bone ESP and aimbot.

Last but not least, Artemis provides a full-fetched extension framework for anyone to use to create and share their own creations with. Whether you are an advanced user, or someone who would just like to try and set foot in the mod-making world of Rainbow Six: Siege, you may find the Artemis Extension Framework to be useful to you.

But what sets Artemis apart?

Project Artemis is aimed to be an official, professional, reliable, and fully open source set of tools delivering clarity to the user through intuitive and easy to use software. This is to make Artemis as broadly accessible as possible, regardless of your amount of experience with old Rainbow Six: Siege.

The current issue with publicly available software for interacting with old versions of Rainbow Six: Siege is their poorly designed user interfaces and confusing ways of working. This both confuses and scares new members of its community. The main goal of Project Artemis is for it to be an exception to this rule.
